FreshStuff is a release bot for Lua-enabled applications. Key features:
- store releases categorized and give them back by category, the latest n ones or all
- release deletion is possible
- request manager module included
- extras module with release pruning and top-release-adders included
- the core and modules are host app independent
- host app specific behaviour is handled by a separate module.
